# 🛡️ Cybersecurity Awareness Chatbot (WPF Desktop App)

This project is a **WPF desktop application** built in **C# using .NET 8**, designed to improve cybersecurity awareness through an interactive chatbot, task assistant, activity log, and a built-in quiz game.

> 📁 GitHub Repository: [ST10207887 Cybersecurity Awareness Chatbot](https://github.com/ST10207887/https-github.com-IIEWFL-prog6221-poe-resubmission-ST10207887.git)

---

## 💻 Features

- ✅ **Chatbot Conversation Engine**  
  NLP-style parsing to respond to cybersecurity-related queries.

- ✅ **Task Assistant**  
  Allows users to add, update, and complete cybersecurity tasks (e.g., "Enable 2FA").

- ✅ **Reminder System**  
  Automatically sets reminders for important tasks.

- ✅ **Activity Log**  
  Logs recent actions like tasks added, reminders set, and quiz activity. Viewable on demand.

- ✅ **Cybersecurity Quiz**  
  A short multiple-choice quiz to test user knowledge. Graded instantly.

- ✅ **Modern GUI**  
  Built using WPF (XAML and C#). User-friendly layout with scrollable chat, task input, and system feedback.
